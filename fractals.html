What it is

Fractals is an interactive visualizer prototype built to explore a bigger idea than audio response alone. The same reactive visual language that makes music feel alive can also support gameplay: combat bursts, boss transitions, menu states, exploration ambience, and player inputs.

In other words, this is both a standalone visual experience and a sandbox for future in-game presentation systems.

Photosensitivity warning: This fractal experience includes flashing and strobe-like visual effects. Please proceed with caution if you are sensitive to rapidly changing light or patterns.

  • Audio-reactive scenes
  • System audio and mic input
  • Fullscreen presentation
  • Prototype for game feedback

Why it matters for games

I want this project to help answer a useful design question: how can visuals behave like a mechanic instead of a wallpaper? A reactive layer can signal danger, reward precision, amplify atmosphere, or reinforce pacing without needing extra text or UI clutter.

That makes Fractals useful as a portfolio piece, but also as R&D for future projects where the screen itself participates in the feel of play.

Best way to use it

  1. Open the visualizer and interact with the controls first.
  2. Use Mic for live input, or System Audio to react to your computer audio when your browser allows it.
  3. Switch to Fullscreen for the intended presentation.
  4. Press F11 to set your web browser to fullscreen for the intended presentation.

Browser security requires a user interaction before microphone, display capture, and fullscreen behavior can begin. That is expected and built into the launch flow.
